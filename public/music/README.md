# Background Music

Bu klasöre `background.mp3` adında bir müzik dosyası ekleyin.

Önerilen:
- Format: MP3
- Süre: 2-5 dakika (loop olarak çalacak)
- Boyut: Maksimum 5MB (web için optimize edilmiş)
- Tür: Sakin, huzurlu, duygusal müzik

Müziği açmak/kapatmak için sağ alt köşedeki müzik butonunu kullanabilirsiniz.
